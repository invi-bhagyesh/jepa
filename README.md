# SYMBA5 — LM-JEPA for Squared Amplitude Calculation

GSoC 2026 evaluation task for [ML4SCI / SYMBA](https://ml4sci.org/gsoc/2026/proposal_SYMBA5.html).

Transformer-based sequence-to-sequence model for predicting squared amplitudes from Feynman diagram amplitudes in QED and QCD 2→2 processes, enhanced with LM-JEPA self-supervised pretraining.

## Tasks Completed

- **Common Task 1.2**: Dataset preprocessing — parsing, index normalization, physics-aware tokenization, train/val/test split
- **Specific Test 2.5**: LM-JEPA pretraining + seq2seq finetuning, baseline comparison, masking ratio ablation

## Running

Run the notebooks in order:

1. `notebooks/01_data_preprocessing.ipynb` — parses raw data, normalizes indices, builds tokenizer, creates train/val/test splits, reports statistics
2. `notebooks/02_jepa_training.ipynb` — JEPA pretraining → seq2seq finetuning (QED + QCD) → baseline comparison → masking ratio ablation

## Model Architecture

- **Encoder/Decoder**: 4 layers, 256 hidden dim, 4 attention heads, 1024 FFN dim (~6-8M total parameters)
- **JEPA pretraining**: Context encoder processes masked input; 2-layer MLP predictor outputs latent predictions; EMA target encoder provides supervision via smooth L1 loss; contiguous span masking at 35%
- **Seq2seq finetuning**: Pretrained encoder + randomly initialized decoder with cross-attention; teacher-forced cross-entropy; beam search (width 5) at inference

## Evaluation Metrics

| Metric | Description |
|---|---|
| Token accuracy | % of correctly predicted tokens (ignoring padding) |
| Sequence exact match | % of predictions matching target sequence exactly |
| Top-5 accuracy | Target appears in top-5 beam search outputs |

Results are reported separately for QED and QCD, with JEPA-pretrained vs. baseline (no pretraining) comparison.
